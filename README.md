# Employee Attendance System

A simple Python + MySQL project to track employee login, logout,
working hours, underworked employees, and daily attendance reports.

## Features
- One login per employee per day
- Duplicate login prevention
- Auto logout at 7:00 PM
- Work hours calculation
- Underwork detection (< 8 hours)
- Daily attendance report
- Logging using Python

## Technologies Used
- Python
- MySQL
- mysql-connector-python
